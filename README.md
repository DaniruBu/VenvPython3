# Виртуальное окружение Python (venv)

##### Активация 
```
~ % cd VenvPython3/
```

```
~ % mkdir venv 
```

```
~ % python3 -m vevn venv
```

```
~ % source venv/bin/activate
```

```
~ % pip install --upgrade pip
```
##### Requirements
```
~ % touch requirements.txt
~ % pip freeze >> requirements.txt
```

##### Деактивация
```
~ % deactivate
```
