# Виртуальное окружение Python (venv)

##### Активация
``` venv\Scripts\activate.bat ``` - для Windows;
```source venv/bin/activate``` - для для Linux и MacOS.

```
~ % cd VenvPython3/
```

```
~ % mkdir venv 
```

```
~ % python3 -m vevn venv
```

```
~ % source venv/bin/activate
```

```
~ % pip install --upgrade pip
```
##### Деактивация
```
~ % deactivate
```
